The instruction of steps to run this project in local machine 

Step 1: Prequisit -> PHP Version 7.3.31, xampp/ laragon

Step 2: Clone the project from github 
        git clone https://github.com/tonmoyGhosh/project_mvc.git

Step 3: Database sql file is in DB folder. Create database & import the file in local machine created   
        database.

Step 4: Config the DB & project url related configuration in app/config/config.php file.
        define('DB_HOST', ''); // Set DB Host like localhost
        define('DB_USER', ''); // Set DB User like root
        define('DB_PASS', ''); // Set DB Password, if any in your local server
        define('DB_NAME', ''); // Set DB Name which you create in your local machine 
        define('Base_url', ''); // Set full path of project as base url

Step 5: To run the project hit the link in your browser http://localhost/project_mvc/index.php
